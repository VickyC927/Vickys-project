專案簡介

本專案結合個人電子發票與信用卡消費資料，並搭配政府家庭收支調查等公開數據， 透過 Power BI 建立互動式視覺化儀表板，分析個人與群體的消費結構差異，探索消費行為背後的風格特徵。

互動展示（Power BI）

分析面向

  1. 消費行為分析
    • 自動分類消費項目（如食、衣、住、行、育樂、其他）
    • 分析消費金額、次數、時間分布、商家偏好
  2. 個人 vs 群體比較
    • 引入全台平均、學生、上班族等族群作對照
    • 計算消費佔比差異，建立比較雷達圖與象限圖
  3. 年度與月份趨勢追蹤
    • 比較不同年份、月份的總花費與消費組成變化
    • 加入月份、星期等時間維度，觀察消費高峰期
  4. 視覺化儀表板
    • 使用 Power BI 製作互動圖表
    • 包含趨勢圖、佔比圖、象限圖與 Tooltip 功能

資料來源

  • 個人電子發票資料（財政部）
  • 信用卡消費明細
  • 政府公開資料（家庭收支調查、物價指數等）

預期效益

  • 認識自身的消費習慣與偏好分類
  • 與群體比較，發現潛在的消費風格特徵
  • 提升財務自覺與資料閱讀能力

使用工具

  • Python（資料前處理、自動分類）
  • SQL / Power Query（資料整合與轉換）
  • Power BI（互動式視覺化）
  • Excel / Google Sheets（初步彙整）

還能擴充的方向？

  • 本次雖以靜態報告為主，但未來仍可進一步開發：
  • 異常消費偵測：出現過高／突增分類時即時提醒
  • 預算超支預警：針對各類別設定門檻並發送通知
  • 智能建議系統：根據消費模式給出節省或分配建議
